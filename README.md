## Start the project

Git clone:

> git clone https://github.com/AllanMont/OC_P5

<!-- Explain how to start Front-end : cd front && npm run start -->
Start Front-End :
> cd front && npm i && npm run start

Start Back-End :
> cd back && mvn spring-boot:run

## Launch tests

Front-End Tests:

> E2E :

>> npx cypress open

> Units :

>> ng test


Back-End Tests:
> mvn clean test
